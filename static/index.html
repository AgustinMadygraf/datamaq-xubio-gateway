DataMaq Xubio Gateway

Bienvenido a DataMaq Xubio Gateway

Este proyecto es un gateway FastAPI para la API de Xubio, usando OAuth2 con credenciales de cliente.

  • Consulta y gestiona clientes y productos de Xubio.
  • Obtén tokens OAuth2 y prueba la integración.
  • Clean Architecture para fácil mantenimiento y extensión.

Para comenzar, selecciona una pestaña en la barra superior.
